## 👤 About Me

I'm a Computer Science student and full-stack developer focused on shipping **complete, production-ready systems** — not demos. My work spans backend architecture, frontend engineering, and increasingly, security: understanding how systems break is part of how I learn to build them right.

- 🏗️ Architecting **Abdou Samy Academy** — a full LMS platform with video streaming, payments, and certification
- 🛒 Building **JOX-STORE** — a real-time digital marketplace with seller dashboards
- ⚔️ Practicing algorithmic problem-solving on **Codeforces**
- 🔐 Studying penetration testing with **Kali Linux, Wireshark, and Burp Suite**
- 🤖 Designing **Telegram bots** with FSM-based conversation logic and webhook infrastructure
- 🌐 Experimenting with **3D web experiences** using Three.js and GSAP
